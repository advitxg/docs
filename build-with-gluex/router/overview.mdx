## Key Features

The GlueX Router API offers several advanced features designed to improve transaction efficiency, reduce risk, and provide transparency to end-users and integrators:

- **Multi-Chain Support**: Seamlessly operates across multiple supported blockchain networks.
- **Comprehensive DeFi Protocol Integration**: Fully supports decentralized exchanges (DEXs), lending platforms, yield farming protocols, staking mechanisms, and more.
- **Optimized Transaction Execution**: Configurations minimize transaction reversions and protect against Maximal Extractable Value (MEV).
- **Rapid Response Times**: Average request round-trip time of the quote is approximately 500 milliseconds.
- **Positive Slippage Rewards**: Any positive slippage is distributed among users, integrators and us.
- **Dynamic, Transparent Fees**: We charge absolutely zero routing fees, ensuring complete transparency with no hidden costs or charges
