The GlueX Exchange Rates API provides developers with a robust, scalable, and real-time solution to fetch on-chain exchange rates for any token pair across multiple supported blockchains. Perfect for DeFi apps, wallets, and trading platforms.

### Key Features

- **Real-Time Rates**: Get accurate, up-to-date rates directly from the blockchain.
- **Multichain Support**: Query rates across Ethereum, BNB Chain, Polygon, and more.
- **Batch Requests**: Retrieve up to 25 token pairs in a single request.
- **No Auth Required**: Public endpoints for instant access, with optional API keys for higher rate limits.
- **Fast & Reliable**: Average request round trip of 500ms.
- **Specialized Support**: Handles unique tokens and edge cases.
